W3C

Widget Packaging and XML Configuration
Standardizes a packaging format and configuration format for client-side web applications.
Widget Interface
Defines APIs to store preferences and access widget package metadata.
XML Digital Signatures for Widgets
Defines a means for widget packages to be digitally signed using a custom profile of the XML-Signature Syntax and Processing Specification.
Widget Updates
Defines a version control model that allows widgets to be kept up-to-date over HTTP.
Widget Access Request Policy (WARP)
Defines a means to request access to URI-identifiable resources (e.g. resources on the Web).
The 'view-mode' Media Feature
Defines a media feature (CSS Media Query) related to the presentation mode of the document's content.
